--- layout: post title: GContracts 1.2.10 Released! categories: - gcontracts - groovy - releases tags: [] status: publish type: post published: true meta: _edit_last: '2' _edit_lock: '1350500176' _wp_old_slug: '' --- I am happy to announce that GContracts 1.2.10 has just been released and is available in the Central Maven repository [0] and at Github [1]. 1.2.10 follows closely to 1.2.9 as it fixes two critical bugs, one involving Groovy 1.8.8, the other Oracle JRE 1.7.

Release Notes

  • [34] ClassNotFoundError when using Groovy 1.8.8
  • [36] Generic parameter type leads to NP when running on Oracle Java 1.7_10
  • [33] Error when constructor parameter and instance variables have the same identifier
  • [35] Introduce inline mode for performance boost
Many thanks to Evgeny Goldin [3] for all his testing and feedback!
